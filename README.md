### 🛒 Supermarket Pricing – JSON Project




## 📌 Project Objective

This project explores product pricing trends across three major UK supermarkets  ( Tesco, Sainsburys and Asda) using over 100,000 product records from multiple categories.This project consist of an EDA ( Exploratory data analysis) aimed to evaluate  pricing strategies, own-brand penetration, product-level value and provide recomendations based on the findings.
